Day 1-2: Introduction to Arrays in Python

Overview of Arrays: Arrays are essential data structures in Python that allow you to store and manipulate collections of items. They provide efficient ways to organize, access, and modify data.

Declaration and Initialization: In Python, arrays are commonly represented using lists. Lists are ordered, mutable, and can contain elements of different data types.

numbers = [1, 2, 3, 4, 5]

Indexing and Accessing Elements: Array elements are accessed by their index. Python uses zero-based indexing, meaning the first element is at index 0.

first_element = numbers[0] # Result: 1

Array Slicing: You can extract portions of an array using slicing. This is useful for working with subsets of data.

subset = numbers[1:4] # Result: [2, 3, 4]

Array Methods: Python provides a variety of built-in methods for manipulating arrays. These include append(), remove(), pop(), and extend().

numbers.append(6) # Adds 6 to the end
numbers.remove(3) # Removes the element 3

Multidimensional Arrays: While lists can contain elements of any data type, you can create multidimensional arrays using lists of lists.

matrix = [[1, 2, 3], [4, 5, 6], [7, 8, 9]]

Example:

# Creating a list of temperatures
temperatures = [25.5, 30.2, 22.8, 28.0, 26.5]

# Accessing the third temperature (index 2)
third_temperature = temperatures[2]
print(third_temperature)  # Output: 22.8

# Modifying the second temperature (index 1)
temperatures[1] = 31.0
print(temperatures)  # Output: [25.5, 31.0, 22.8, 28.0, 26.5]

Understanding these concepts will empower you to work with arrays effectively in Python. Practice with different scenarios to solidify your understanding.
